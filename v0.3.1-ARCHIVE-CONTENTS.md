## 🚀 Использование архива

### Распаковка
```bash
cd ~/.openclaw
tar -xzf memory-v0.3.1.tar.gz
```

### Установка зависимостей
```bash
cd memory
npm install
```

### Проверка
```bash
npm test                    # Все тесты
npm run stats              # Статистика
npm run query "python"     # Поиск
```
