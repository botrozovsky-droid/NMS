## ⚙️ Настройки (если нужно)

Файл: `~/.openclaw/memory/meta/learning-params.json`

**Основные параметры:**

```json
{
  "consolidation": {
    "minImportance": 0.3,      // Порог важности (0-10)
    "schedule": "0 3 * * *"    // Ночная консолидация в 3:00
  },
  "session": {
    "inactivityThreshold": 1800000,  // 30 минут
    "minEventsForConsolidation": 3   // Минимум событий
  }
}
```

**Что можно настроить:**
- `minImportance` - какие события считать важными
- `inactivityThreshold` - когда заканчивается сессия
- `minEventsForConsolidation` - минимум для консолидации

## ❓ Частые вопросы

### Q: Нужно ли что-то делать для активации?
**A:** Нет! Система уже работает автоматически.

### Q: Где хранятся данные?
**A:** Локально в `~/.openclaw/memory/`
- Безопасно (не в облаке)
- Приватно (только у вас)
- Быстро (локальный доступ)

### Q: Сколько это стоит?
**A:** $0.58/месяц (цена чашки кофе ☕)
- В 50-100 раз дешевле конкурентов!
- Никаких скрытых платежей

### Q: Можно ли отключить?
**A:** Да, но зачем? AI станет "забывчивым" 😊
Если нужно: удалите данные из `~/.openclaw/memory/`

### Q: Безопасна ли система?
**A:** Да!
- ✅ Транзакции защищают от потери данных
- ✅ Автоматический откат при ошибках
- ✅ Данные только у вас (не в облаке)
- ✅ Нет записи паролей и токенов

### Q: Что если AI "запомнит" что-то неправильно?
**A:** Просто скажите: "Это неправильно" и AI исправит память.
Система учится на исправлениях!
