## 🚀 Как использовать

### Для вас (пользователь):

**Просто работайте с AI как обычно!**

Система памяти работает **автоматически**:
- AI запоминает ваши предпочтения
- AI понимает контекст проектов
- AI не повторяет ошибки
- AI адаптируется под ваш стиль

### Для AI OpenClaw:

AI автоматически использует память через:
```javascript
// Инициализация (автоматически)
await memoryManager.initialize();

// Поиск контекста (автоматически)
const context = await memoryManager.queryKnowledgeGraph(query);

// Запись событий (автоматически)
await memoryManager.encodeEpisode(sessionId, event);
```
