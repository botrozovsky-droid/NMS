## ✨ Что нового

### 1. 📓 Obsidian Export
Экспорт графа знаний в Obsidian-совместимый Markdown:
- Bidirectional wikilinks `[[concept]]`
- YAML frontmatter с метаданными
- Автоматический INDEX.md со статистикой
- Организация по типам (concepts, events, procedures)

**Команда:** `npm run export:obsidian`

### 2. 🔍 Contradiction Detector
Автоматический поиск противоречий между фактами:
- Gemini API для семантического анализа
- Стратегии разрешения (prefer_newer, prefer_higher_confidence)
- Автоматическое снижение confidence у противоречивых фактов
- Batch processing для масштабирования

**Команды:**
- `npm run check:contradictions` - проверить
- `npm run fix:contradictions` - проверить и исправить
