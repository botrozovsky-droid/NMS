# OpenClaw Memory System v0.2 - Technical Specification

**Version:** 0.2.0
**Date:** 2026-04-11
**Status:** Implementation in progress
**Strategy:** Hybrid Enhancement (Metadata + Dual-Source)

---

## 📋 Overview

Version 0.2 enhances the neurobiological memory system with:
1. **Graphify-inspired metadata** (extractionType, confidence, rationale)
2. **Dual-source architecture** (code + experience)
3. **Canonical forms** for deduplication
4. **Hallucination detection** (automatic flagging)
5. **Improved transparency** (full audit trail)

**Core Philosophy:** Combine the best of both worlds:
- **Graphify:** Precision, transparency, determinism
- **Our approach:** Adaptation, learning, temporal dynamics

---

## 🎯 Key Objectives

### 1. Enhanced Transparency
**Problem (v0.1):** Can't distinguish facts from inferences.

**Solution (v0.2):** Every node/edge tagged with:
- `extractionType`: `EXTRACTED` | `INFERRED` | `HYBRID`
- `confidence`: 0.0-1.0 (certainty level)
- `rationale`: Human-readable explanation with evidence

### 2. Automatic Deduplication
**Problem (v0.1):** LLM creates synonyms ("config.json", "configuration", "json_config").

**Solution (v0.2):** Canonical forms system:
- Each node has `canonicalForm` field
- Aliases automatically merged into canonical
- Fuzzy matching for detection

### 3. Hallucination Detection
**Problem (v0.1):** Gemini may infer connections not in evidence.

**Solution (v0.2):** Automatic flagging:
- Track `evidenceEvents` for each claim
- Flag when `INFERRED` + no evidence + low confidence
- Allow manual correction with learning

### 4. Dual-Source Architecture
**Problem (v0.1):** Cold start (2-4 weeks), no code awareness.

**Solution (v0.2):** Two sources per node:
- `codeSource`: Extracted from code (if available)
- `experienceSource`: Learned from events (always)
- Merge for complete picture

## 🔄 Migration Strategy

### Automatic Migration Script

**File:** `migrate-v0.1-to-v0.2.js`

**Process:**
1. Read existing v0.1 graph
2. For each node/edge:
   - Add default metadata:
     - `extractionType: "INFERRED"` (all v0.1 data is LLM-inferred)
     - `confidence: 0.7` (moderate default)
     - `rationale: "Migrated from v0.1 - extracted by Gemini"`
     - `evidenceEvents: sources` (use existing sources)
   - Add canonical form:
     - `canonicalForm: normalize(node.id)` (lowercase, no spaces)
     - `aliases: []` (empty initially)
   - Add dual-source:
     - `codeSource: null` (not available in v0.1)
     - `experienceSource: {...}` (populate from existing data)
   - Add flags:
     - `flags: []` (empty initially)
3. Save to same file (v0.1 backed up)
4. Validate migrated structure

**Backward Compatibility:**
- All v0.1 fields preserved
- Old code can still read basic fields (id, weight, type)
- New code reads extended fields

### Response Processing

**Enhanced parsing** in `consolidate.js`:

```javascript
async function processGeminiResponse(response) {
  const analysis = JSON.parse(response);

  // Validate metadata
  for (const concept of analysis.concepts) {
    if (!concept.extraction_type || !concept.confidence || !concept.rationale) {
      throw new Error(`Missing metadata for concept: ${concept.name}`);
    }

    // Validate confidence range
    if (concept.confidence < 0 || concept.confidence > 1) {
      throw new Error(`Invalid confidence for ${concept.name}: ${concept.confidence}`);
    }

    // Auto-flag potential hallucinations
    if (concept.extraction_type === "INFERRED" &&
        concept.confidence < 0.5 &&
        !concept.rationale.includes("episode")) {
      concept.flags = ["potential_hallucination"];
    }
  }

  return analysis;
}
```

## 🛠️ New Utility Modules

### 1. canonical-forms.js

**Purpose:** Deduplication through canonical forms

**Functions:**
```javascript
// Normalize node name to canonical form
function canonicalize(name) {
  return name.toLowerCase().replace(/[^a-z0-9]/g, '');
}

// Find potential duplicates
function findDuplicates(graph) {
  const canonical = {};
  const duplicates = [];

  for (const [nodeId, node] of Object.entries(graph.nodes)) {
    const canon = node.canonicalForm || canonicalize(node.name);
    if (canonical[canon]) {
      duplicates.push({ original: canonical[canon], duplicate: nodeId });
    } else {
      canonical[canon] = nodeId;
    }
  }

  return duplicates;
}

// Merge duplicate nodes
function mergeDuplicates(graph, canonical, duplicate) {
  const canonNode = graph.nodes[canonical];
  const dupNode = graph.nodes[duplicate];

  // Merge activations, weight (weighted average)
  canonNode.activations += dupNode.activations;
  canonNode.weight = (canonNode.weight * canonNode.activations +
                      dupNode.weight * dupNode.activations) /
                     (canonNode.activations + dupNode.activations);

  // Merge aliases
  canonNode.aliases = canonNode.aliases || [];
  canonNode.aliases.push(dupNode.name, ...(dupNode.aliases || []));

  // Merge evidence
  canonNode.evidenceEvents = [
    ...(canonNode.evidenceEvents || []),
    ...(dupNode.evidenceEvents || [])
  ];

  // Delete duplicate
  delete graph.nodes[duplicate];

  // Update edges pointing to duplicate
  for (const edge of Object.values(graph.edges)) {
    if (edge.source === duplicate) edge.source = canonical;
    if (edge.target === duplicate) edge.target = canonical;
  }

  return canonNode;
}
```

### 2. hallucination-detector.js

**Purpose:** Automatic detection of potential hallucinations

**Functions:**
```javascript
// Detect potential hallucinations
function detectHallucinations(graph) {
  const flagged = [];

  for (const [nodeId, node] of Object.entries(graph.nodes)) {
    const flags = [];

    // Rule 1: INFERRED + no evidence + low confidence
    if (node.extractionType === "INFERRED" &&
        (!node.evidenceEvents || node.evidenceEvents.length === 0) &&
        node.confidence < 0.5) {
      flags.push("potential_hallucination");
    }

    // Rule 2: Code source exists but never used in practice
    if (node.codeSource?.exists &&
        node.experienceSource?.activations === 0) {
      flags.push("unused_import");
    }

    // Rule 3: High confidence but no evidence
    if (node.confidence > 0.8 &&
        (!node.evidenceEvents || node.evidenceEvents.length === 0)) {
      flags.push("suspicious_confidence");
    }

    if (flags.length > 0) {
      node.flags = [...(node.flags || []), ...flags];
      flagged.push({ nodeId, flags });
    }
  }

  return flagged;
}

// Validate edges for hallucinations
function validateEdges(graph) {
  const invalid = [];

  for (const [edgeId, edge] of Object.entries(graph.edges)) {
    // Rule: INFERRED relationship should have evidence or rationale
    if (edge.extractionType === "INFERRED" &&
        (!edge.evidenceEvents || edge.evidenceEvents.length === 0) &&
        (!edge.rationale || edge.rationale.length < 20)) {
      edge.flags = edge.flags || [];
      edge.flags.push("weak_inference");
      invalid.push(edgeId);
    }
  }

  return invalid;
}
```

### 3. confidence-analyzer.js

**Purpose:** Compute composite confidence scores

**Functions:**
```javascript
// Compute final confidence from multiple sources
function computeConfidence(node) {
  let confidence = 0;
  let weight = 0;

  // Code source (high weight)
  if (node.codeSource?.exists) {
    confidence += node.codeSource.confidence * 0.6;
    weight += 0.6;
  }

  // Experience source (medium weight)
  if (node.experienceSource) {
    confidence += node.experienceSource.confidence * 0.4;
    weight += 0.4;
  }

  // Direct extraction (override)
  if (node.extractionType === "EXTRACTED" && node.evidenceEvents?.length > 0) {
    return Math.max(0.9, confidence / weight);
  }

  return weight > 0 ? confidence / weight : 0.5;
}

// Adjust confidence based on activations (Bayesian update)
function updateConfidenceWithUsage(node) {
  const baseConfidence = node.confidence;
  const activations = node.activations || 0;

  // More activations = higher confidence (evidence of existence)
  const usageFactor = Math.min(1.0, Math.log(activations + 1) / Math.log(50));

  // Bayesian update: P(true | usage) ∝ P(usage | true) * P(true)
  const updatedConfidence = baseConfidence + (1 - baseConfidence) * usageFactor * 0.2;

  return Math.min(1.0, updatedConfidence);
}
```
